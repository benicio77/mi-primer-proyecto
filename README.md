# mi-primer-proyecto

Este es mi primer proyecto en Inteligencia Artificial.

## Empezando

1. Install dependencies
   - Example (Node): `npm install`
   - Example (Python): `python -m venv .venv && source .venv/bin/activate && pip install -r requirements.txt`

2. Run locally
   - Example (Node): `npm start`
   - Example (Python/Flask): `flask run`

3. Run tests
   - `npm test` or `pytest`

## Development

- Create issues for new tasks and assign labels/milestones.
